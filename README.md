# VisionXAID

VisionX AI is a Streamlit-based web application that uses a deep learning model to analyze medical images and provide AI-powered predictions.

## Overview

The application allows users to upload medical images and receive automated predictions generated by a trained neural network model. It is designed with a simple and interactive user interface to make AI-based analysis accessible and easy to use.

## Key Features

- Medical image upload functionality
- AI-driven disease prediction
- Confidence score display
- Clean and interactive Streamlit interface
- Structured and modular code design

## Tech Stack

- Python
- Streamlit
- TensorFlow / Keras
- NumPy
- Pillow

## Project Structure

VisionXAid/  
│── app.py # Main Streamlit application  
│── script.py # Model loading and prediction logic  
│── requirements.txt # Project dependencies  
│── assets/ # UI assets
